Ubuntu18.04和WIN10双系统安装

今天讲一下Ubuntu的安装，Ubuntu的安装网上一搜一大堆，本文讲和WIN10双系统安装，单独装Ubuntu也可参考，比双系统安装更简单。目前网上大多数教程是在虚拟机下安装的，但为了后续学习，还是直接上双系统，我的电脑是一个固态和一个机械盘，因此计划在固态盘装WIN10,在机械盘装Ubuntu。

WIN10安装

WIN10系统的安装本文不讲，很简单，制作个安装盘，直接装好了，装好后激活下就可以了，网上搜索下就行。需要注意的是现在最新的系统安装都是UEFI+GPT安装了，传统的BIOS+MBR模式后面会逐渐淘汰，这对后面安装Ubuntu的兼容性更好。电脑在BIOS里设置UEFI模式，另外如果有secure boot模式则要关闭掉。UEFI是引导模式，GPT和MBR是分区类型，这要在新建磁盘的时候设置，当然也可以后面转，但这有可能会数据丢失或者出错。
下面是我的安装方式，前面用win10安装盘进入安装界面后，在进入后在磁盘界面后按shift+F10进入命令符：

  1. Diskpart （进入分区管理）
  2. list disk （列出所有磁盘）
  3. select disk X （选择进入的磁盘， 选择你的win10要按在那个盘里， 一般是 0）
  4. clean （ 第三步完成 会提示 所选磁盘，这一步是清楚磁盘，数据会被清楚）
  5. convert GPT （转换成GPT 格式）
    完成后，关掉窗口，选择磁盘安装WIN10，一路装完就可以
    要注意的也就这些，网上找个版本下了装就可以了，装好百度激活下。

激活方法：

  1. 开始菜单右键进入windows powershell（管理员）
  2. 输入slmgr /ipk 神Key，这个神Key百度搜索，搜索的时候注意搜索时间设为一周最近的
  3. slmgr /skms kms.xspace.in设置激活名称
  4. slmgr /ato激活命令
  5. slmgr.vbs -xpr查看是否永久激活

Ubuntu安装

介绍

Ubuntu（友帮拓、优般图、乌班图）是一个以桌面应用为主的开源GNU/Linux操作系统，Ubuntu 是基于Debian GNU/Linux，支持x86、amd64（即x64）和ppc架构，由全球化的专业开发团队（Canonical Ltd）打造的。
Ubuntu有很多个版本，有Desktop、Server、Iot、Cloud版本分别是桌面版、服务器版、物联网版和云版。一般我们选择桌面版就可以，其实后来了解到桌面版和服务器版其实没有什么区别，只是默认安装的包不同，这个在系统安装好后，还可以根据需要下载安装自己需要的包。为什么装桌面版，因为桌面版默认带我们熟悉的桌面操作环境，而服务器版如果要桌面环境还要另外安装。我们用惯了WINDOWS系统，如果一个系统上来就只有命令行，我们很难想象，所以还是装桌面版比较适合我们，后面如果有一些服务器版的功能需要，再另外安装就好了。
在官方界面里面还有有LTS版本和其它的版本选择。不带LTS的版本往往比较新，这个有点类似与尝鲜版，不稳定，我们一般选择安装LTS版本，叫做长期支持版。目前最新的版本为18.04。我们下载安装就好了。

下载

Ubuntu的官网是https://ubuntu.com/index_kylin，大家登录选择DOWNLOAD页签进去后，就能看到各个下载版本，界面很简单，不同英文应该也能看懂。后面要习惯看英文网站和官网，因为有一些问题可以从官网上找到答案。

制作安装盘

我刚开始是用UltralISO做的，很简单：

  1. 打开镜像文件
  2. 选择启动下的写入硬盘镜像
  3. 在对话框里选择要做的U盘，格式用USB-HDD+
  4. 然后格式化，格式化完成后写入就可以了
    后来重新用Universal-USB-Installer这个软件做的。
    步骤很简单：
  5. 选择你要安装的系统Ubuntu
  6. 选择你下载的Ubuntu.iso文件
  7. 选择你的U盘
    然后就好了。

分出空闲空间

首先，在现有计算机下划出一个专门给Ubuntu的磁盘分区，推荐Windows自带的磁盘管理工具。快捷键“win+X”->“磁盘管理”->“压缩卷”（Windows安装在固态硬盘则压缩固态硬盘，机械硬盘同理），压缩完之后会多出了一块“未分配空间”给Ubuntu。下图是个例子，盗了个人家的图^^。

安装

开始正式安装，网上有说要先进入BIOS里面把Secure Boot这个功能给关掉，好像设计初衷是安全保护，但后来变为厂家为了垄断，不让用户安装其它系统用。具体也没去试不关掉有什么问题，为了后面安装不出问题就去关掉吧，如果BIOS没有这个选项就跳过。

关闭计算机，插入已经做成启动盘的U盘，进入Boot menu选择界面，选择“Istall Ubuntu”，图形化的安装界面，建议用英文模式安装，安装完后再进系统改语言，改的时候不要改文件名称，这是为了以后装一些软件不出问题，因为Ubuntu的中文语言会将相应的文件路径也改成英文。软件暂时不要安装，网也不要连，选典型安装，继续，当出现要手动分盘符的时候，请选择“其它选项”。
找到预先留出的 free space 空间起始位置不用动，

  • 双击选定大小为500M的 逻辑分区 EFI 格式 EFI 引导
  • 选定8G左右 主分区 设定为 swap 交换空间
  • 选定10G+左右 主分区 ext4 挂在到 ‘ / ’ 下
  • 剩下的 主分区 ext4 挂载到 ‘ /home ’ 下
    注意不需要挂在/boot 不需要挂在/boot 注意不需要挂在/boot 重要的事情说三遍！！
    最后需要将安装启动引导器的设备这一栏更改为刚才分的EFI分区，比如我的是/dev/sdb5
    选择完毕后，点击继续，后面的就是简单的了，有让你确定地址，直接默认继续，然后让你输入名字，密码什么的，自己设置一下就好；跟着步骤都点完后，就可以点击安装了，安装完毕会提示重启，点击重启就好（U盘可以不拔掉，但是需要进入BIOS界面，会是UEFI引导启动，找到Ubuntu启动项，点击启动就好，会出现让你选择Ubuntu还是Win10系统，自己选择吧。）
