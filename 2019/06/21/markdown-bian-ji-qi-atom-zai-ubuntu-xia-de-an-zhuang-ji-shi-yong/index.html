Markdown编辑器Atom在Ubuntu下的安装及使用

作为一个从没发过博客新手，突然萌生了要把日常中的点点滴滴记录下来的念头，结果一发不可收拾，连续几个晚上，安装了Ubuntu系统，搭建了了Apache2服务，最后安装了Hexo，中间遇到了很多问题，更加坚定了要做博客把这些记录下来的想法。下面就从安装和使用Markdown编辑器Atom开始我的博客之旅吧！

Atom简介

工欲善其事，必先利其器！写博客，必须选择一个好的Markdowm编辑器，网上关于这方面的介绍很多，眼花缭乱，各人有各人的喜好。综合了个方面介绍，最终选择了Atom。为什么会选它呢？先来看下百度百科的介绍。
Atom 是github专门为程序员推出的一个跨平台文本编辑器。具有简洁和直观的图形用户界面，并有很多有趣的特点：支持CSS，HTML，JavaScript等网页编程语言。它支持宏，自动完成分屏功能，集成了文件管理器。
看中了两点，一是有github推出的，二是开源。其它各种功能暂且不谈，毕竟不可能玩的花里胡哨的，有基本的功能就行。

安装

Atom的安装很简单

$ sudo add-apt-repository ppa:webupd8team/atom
$ sudo apt-get update
$ sudo apt-get install atom

atom的安装过程很慢，中间下载失败了几次，应该是网速的问题，多试几次install atom就好了，会接着上次失败的继续下载的。

卸载

卸载没用试过，先记录下方法吧，以后说不定用的到

$ sudo apt-get remove atom
$ sudo add-apt-repository --remove ppa:webupd8team/atom

以上只会卸载该软件，要卸载附加的一些软件包，请使用以下命令卸载多余的软件包

$ sudo apt-get autoremove

开始写博客

首先还是在刚才的welcom guide中点击open a project,然后选择你的md文件存放的目录就可以，然后在窗口的最左边的project窗口中显示这个文件夹下所有的文件了，一旦你在那个文件下创建的了md文件，这个窗口中就会显示出相应的文件了，点击你要编辑的文件，就可以打开对应的编辑窗口，然后就可以开始编写你的博客了，在编写过程中按下ctrl+shift+m就可以预览博客效果了。
