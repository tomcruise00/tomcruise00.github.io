ROS简介

历史

2007 诞生于斯坦福STAIR项目
2010 ROS 1.0发布
2014 ROS Ingigo发布
2016 ROS Kinetic发布
2017 ROS 2.0Ardent发布
2018 ROS Melodic发布

特点

  1. 点对点的设计
    1. 节点单元
    2. 分布式网络
    3. RPC+TCP/UDP通用系统
    4. 适合多机协同
  2. 多语言支持
  3. 架构精简、集成读高
  4. 组建化工具包丰富
    1. 3D可视化工具——rviz
    2. 物理仿真环境——gazebo
    3. 数据记录工具——rosbag
    4. Qt工具箱——rqt_*
  5. 免费并且开源

ROS系统实现——计算图

  1. 节点（Node）——软件模块
  2. 节点管理器（ROS Master）——控制中心，提供参数管理
  3. 话题（Topic）——异步通行机制，传输消息
  4. 服务（Service）——同步通信机制，传输请求/应答数据

ROS安装

直接到ROS wiki上去查看如何安装，因为版本更新比较快，每个版本如何安装还是直接看官方文档说明。因为最新的Ubuntu已经是18.04了，所以选择ROS的LTS版本是melodic的版本，安装说明网址是http://wiki.ros.org/melodic/Installation/Ubuntu，注意要看英文网站，中文网站没有这个版本。

更新安装源

sudo sh -c 'echo "deb http://packages.ros.org/ros/ubuntu $(lsb_release -sc) main" > /etc/apt/sources.list.d/ros-latest.list'

添加密钥

sudo apt-key adv --keyserver 'hkp://keyserver.ubuntu.com:80' --recv-key C1CF6E31E6BADE8868B172B4F42ED6FBAB17C654

安装ROS

sudo apt update
sudo apt install ros-melodic-desktop-full   //这是完全安装,这个时间比较长，根据网速决定

初始化rosdep

sudo rosdep init
rosdep update

设置环境变量

echo "source /opt/ros/melodic/setup.bash" >> ~/.bashrc
source ~/.bashrc

这里的bash是根据自己用的终端来改的，我用的zsh，所以这里要改成zshrc

安装一些功能包

sudo apt install python-rosinstall python-rosinstall-generator python-wstool build-essential

启动

在终端输入roscore启动ROS

roscore

查看计算图

rqt_graph

参数跟踪工具

rqt_plot

ROS常用命令

rosnode
rostopic
rosservice

输入ros在按TAB键可以看到所有的ros命令;
在输入命令后输入空格再按TAB可以看到这个命令下的命令;
在输入命令后输入–help可以查看相应的命令的帮助。
